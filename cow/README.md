Все данные ниже команды нужно запускать из корневой директории директории модуля ```/cow```
# Использование
Запуск демонстрации работы интерпретатора на примере [hello.cow](cow_examples/hello.cow) с выводом "Hello, World!":
```
go run main.go
```

# Тестирование и coverage
## Запуск тестов с подробным выводом в терминал
```
go test ./interpreter -v
```
## Составление html-отчёта покрытия
Запись в файл ```сover``` исполненных во время тестирования строк кода тестируемого пакета для последующего анализа с ```go tool cover```:
```
go test ./interpreter -coverprofile=cover
```
Составление отчёта покрытия в формате html по файлу ```cover```:
```
go tool cover -html=cover

```
